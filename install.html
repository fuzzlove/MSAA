Install MSAA

Install macOS Security Audit Agent from PyPI.

Run local-first macOS security audit, monitoring, intrusion detection, and investigation workflows without a cloud dependency by default.

pip install macos-security-audit-agent

Run the application

After installation, start MSAA from the command line. Some advanced monitoring features may require macOS permissions or administrator approval.
